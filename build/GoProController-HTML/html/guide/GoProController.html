GoProController Package

GoProController is a library of functions used to control GoPro cameras. It contains many functions to set every possible parameter of camera such as resolution, FPS, FOV, etc. It also contains functions for downloading recorded content from camera. Get to know your camera well. You need to know which parameters can be set for each setting of camera. There are functions which will help you to choose right parameters but they are not unmistakable. This package was tested only on GoPro HERO 3+ Black edition and HERO4 Silver edition so it may contain bugs for other cameras.


Functions listed bellow are crucial for this package to work right and you will have to call them before trying controlling the GoPro camera.
Functions listed bellow are crucial for this package to work right and you will have to call them before trying controlling the GoPro camera.

goProSetCamera — function for setting camera model. You have to set camera model before using this package further.

goProGetPossibleCamera — function returning List of possible models for function goProSetCamera.
goProSetCameraModel — function for setting further specification of camera model. That has to be set to make this package fully work.
goProGetPossibleCameraModel — function returning List of possible models for function goProSetCameraModel
goProSetPassword — function which will let you set password for WiFi which camera distributes. You will have to set password if you want to use HERO3 or HERO3+.


Functions listed bellow control GoPro camera.
Functions listed bellow control GoPro camera.

goProSet — function for setting every possible setting of camera via Options.
goProGet — lets you see how the camera is set.
goProTurnOn — turns on the camera.
goProTurnOff — turns off the camera.
goProShutter — camera starts recording or takes a photo.
goProStop — camera stops recording.
