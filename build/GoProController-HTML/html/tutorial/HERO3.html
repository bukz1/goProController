Package setup

Before controlling camera you need to set the package first. You have to do that because package need to know which files have to be loaded and it is impossible to control goProCamera without WiFi password.

goProSetCamera[camera]Sets camera generation to parameter given. You can use goPoGetPossibleCamera to see which cameras are supported by this package.
goProGetPossibleCamera[]returns possible cameras for this package
goProSetPassword [password_String]you need to call this function to set camera WiFi password. Without that package can't control your camera.
goProSetCameraModel[model]Specificates camera model in generation (Black, White,..). The package needs this information.
goProGetPossibleCameraMode[]returns possible camera models within generation

Setting camera parameters

For setting camera parameters we use functions which names starts on string "goProSet ". Those functions can set only one parameter of camera at time. For example function goProSetVideoRes[_String] will set video resolution of camera.

The second way to set camera is to use function goProSet. This function will let us specify many parameters of camera at once and it works through Options. The default value for every option is the value on which is camera currently set. If it wasn't this way camera would be set to default options with every usage of this function (except the options used in its use).

Controlling the camera

We can control camera via functions for shutter and for changing camera mode.

goProTurnOn[]turns on camera
goProTurnOff[]turns off camera
goProShutter[]starts recording or take a photo (depends on camera mode)
goProStop[]stops recording
goProMode[mode]sets camera mode to parameter given
goProGetPossibleModes[]returns possible modes on camera
goProVideoMode[]switches camera to video mode
goProPhotoMode[]switches camera to photo mode
goProBurstMode[]switches camera to burst mode
goProTimeLapseMode[]switches camera to Time lapse mode

Downloading and deleting files from camera

The downloading files from camera is quiet simple. It is possible to download it manually from http://10.5.5.9:8080/DCIM/ or by using functions from this package.

There are three functions that are made to make downloading easy. Functions goProGetURLBase and goProSetURLBase will let you know or let you set the URL from which data will be downloaded. The default value is http://10.5.5.9:8080/DCIM/100GOPRO/

The function goProGetFileList will return list of files at the address which is set by goProSetURLBase or at default URL ( http://10.5.5.9:8080/DCIM/100GOPRO/ ).

Finally function goProDownloadFile will let you download file which name you give as parameter. You can use it for downloading single file or list of files. To get names of files you can use function goProGetFileList. Function goProDownloadAllFiles downloads all files from camera which are in folder set by goProSetURLBase.

goProSetURLBase [url_String]lets you set url of folder from which you want to download files. Default value is http://10.5.5.9:8080/DCIM/100GOPRO/
goProGetURLBase[ ]returns String of url from which files will be downloaded
goProDownloadFile[name,dest]downloads file named name from camera to the destination dest
goProDownloadAllFiles [dest]download all files from camera to destination given
goProGetFileList []returns List of file names present in folder which you get by goProGetURLBase
goProGetFileURL[name]returns string of whole URL to the file which name is given as parameter

Functions for downloading files from camera.

For deleting files we have two functions. One for deleting all files from camera and other for deleting only last file captured.

goProDeleteLast[ ]deletes last file taken on camera
goProDeleteAll[ ]deletes all files on camera
