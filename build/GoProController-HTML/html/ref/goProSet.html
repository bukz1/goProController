goProSet

goProSet[setting1value1,setting2value2,...]     sets GoPro setting1,setting2 to Span[{class -> TI}, value1, ,, value2, , ...] to get list of setting parameters use

goProGetVariables
. goProSet stands for single functions such as goProSetVideoRes, goProSetFPS, etc. It calls this functions in its body with parameters given.

DetailsDetails

  • Function goProSet enables to set many GoPro settings at once. It is based on Options function and its options you can find by goProGetVariables which will also enable whispering on those variables. goProSet itself calls other functions in its body such as goProSetVideoRes, goProSetFPS, etc.
