goProSetPassword

goProSetPassword[param_String]
function which will let you set password for WiFi which camera distributes. You will have to set password if you want to use HERO3 or HERO3+.

DetailsDetails

  • The password for the WiFi must be set before using this package for GoPro HERO3 or HERO3+. That is because the password is needed for controlling the camera.
