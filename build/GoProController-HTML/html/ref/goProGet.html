goProGet

goProGet[param_]
returns List {param -> value} where value is setting of camera which name param holds.

goProGet[{param1_,param2_,...}]
returns List {param1 -> value1, param2 -> value2, ...} where value1 is setting of camera which name param1 holds. Param can be String or name of variable from goProGetVariables.
