goProGetPossibleCameraModel

goProGetPossibleCameraModel[]
returns List of camera models supported by the camera you selected via goProSetCamera. Such as Black, Silver, White, Session, etc. This function is usable for goProSetCameraModel which needs to be set for package working correctly.
