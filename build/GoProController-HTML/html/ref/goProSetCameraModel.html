goProSetCameraModel

goProSetCameraModel[param]
sets camera model to parameter param to further define model which you set via function goProSetCamera, such as Black, Silver, White, Session, etc. This parameter will be used further in using GoProController package.

DetailsDetails

  • The need of this function is determined by various setting for each camera model. For example White edition has fewer setting options than Black edition and so on.
